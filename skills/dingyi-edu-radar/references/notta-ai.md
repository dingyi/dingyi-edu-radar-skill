# 前言介绍

Notta是一款集成了最新AI语音识别引擎的高精度文字转写服务，支持 104 种语言。您可以通过录制音频和上传文件来获得自动转录。它不仅提供实时文字转写和翻译功能，还能快速将长达5小时的音频文件转换成文本。Notta适用于处理会议记录、访谈和各种音频资料，非常适合个人和企业用户使用。

Notta 与 Slack、Salesforce 和 Google 日历等流行的协作工具无缝集成，通过方便的共享和协作会议笔记来提高生产力。用户可以通过网页应用、移动应用或 Chrome 扩展程序访问 Notta，确保使用该服务的灵活性。Notta 致力于数据安全，采用企业级措施保护用户信息。总体而言，Notta 为希望自动化笔记记录和提高会议效率的人士提供了一个全面的解决方案。

# 常见问题

**什么是“工作区”？**

Notta 中的工作区是一个集中枢纽，个人或团队可以从任何设备访问、组织或搜索录音和转录内容。 如果您使用的是商业/企业计划，则同一团队工作区中的成员可以查看和协作其他成员创建的内容。

**你们支持哪些付款方式？**

您可以在 Notta.ai 网站上使用信用卡或借记卡付款，或通过您的 Apple 或 Google 账户进行应用内购买。对于企业套餐订单，我们也接受银行转账。支持的卡类型有：美国运通、Discover、万事达、Visa、JCB

**我如何更改我的订阅计划？**

如果您想从月度计划切换到年度计划，您可以直接在计费部分切换到年度计划。 但是，如果您想从 Pro 切换到 Business，或者从年度计划切换到月度计划，则需要取消当前计划并订阅新计划。 注意：当前计划将一直有效，直到下一个续订日期，新计划将从下一个计费期开始生效。

**你们为学生或教育工作者提供折扣吗？**

是的，Notta 很乐意为教育工作者和学生提供 Notta Pro 计划的折扣。请使用您的教育邮箱注册，并在购买时享受折扣。 了解有关 如何申请教育折扣的 更多信息 。
