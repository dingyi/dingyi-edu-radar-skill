# 常见问题

https://www.ikonpass.com/en/faq

**SheerID是什么？**

SheerID 在购买结帐过程中验证当前的军人、学生和护士身份。购买任何 Ikon Pass 时，仅在准备进行最终购买时使用 SheerID 一次来验证状态。 （尝试次数过多会阻止购买，以防止欺诈。）如果参与机构中未列出护士的医院/诊所，则护士应使用其州许可委员会编号。

**护士、大学生或军人有折扣吗？**

当 Ikon Pass 可供购买时，现役持照护士、在校大学生以及北美现役和退役军人（由 SheerID 验证）可享受折扣。对于军事人员，成年家属也可享受折扣。折扣不能合并。

**哪些产品有资格获得延期信用？**

未使用的 Ikon Pass、Ikon Base Pass、Ikon Base Plus Pass 和 Ikon Session Pass 均有资格获得延期积分。
[细节有所不同。](https://www.ikonpass.com/deferral-terms-and-conditions)
。

**我可以在下一季使用我的积分购买价格较低的 Ikon Pass 产品吗？**

是的。但是，如果积分金额与购买的 Ikon Pass 之间存在价格差异，则不会发放部分积分，也不会进行现金退款。任何剩余余额将被没收。积分不可转让，且仅适用于与原始 Ikon Pass 持有者同名的单张 Ikon Pass。

**我可以将我的积分用于购买个人目的地通票或门票产品吗？**

不可以。您的积分只能用于购买单个未来的 Ikon Pass，而不能用于购买单个 Ikon Pass 目的地产品。

**Ikon Pass 是否为通票持有者提供缆车票、住宿、零售、租赁、滑雪学校或餐饮的额外折扣？**

**Ikon Pass 应用程序是什么？**
