# 前言介绍

在生命科学和医学基础研究中，或多或少都需要用到抗体。抗体的应用场景不同、标记物众多，且针对同一标记物市场上往往有多个抗体可供选择，我们需要从成百万上千万种抗体中选中自己需要的那支，因此citeab就出现了。

CiteAb（www.citeab.com） 是由英国Bath大学的Andrew Chalmers博士于2014年主导创立的，其号称是世界最大的抗体搜索引擎，该网站上包含了来自204个抗体供应商的近五百万个抗体，汇总了两百多万条抗体引用数据，并且该网站仍在不断更新最新的数据。输入关键词（如：CD3）后点击搜索即可出现所有符合条件的抗体，该平台目前包括来自335个供应商的699万种抗体，引用数量超过424万。而该网站的特色是，搜索结果根据抗体被文献引用的次数排序，一般来说，引用越多的产品，质量往往越有保证。citeab网站界面简洁，不仅可以搜索抗体，还可以查询生化试剂、实验方法、试剂盒以及蛋白。

# 使用方法

我们以常用内参β-actin为例进行CiteAb搜索。在搜索栏输入查询抗体名称“β-actin”（只支持英文查询），在搜索栏后方的目录中选择“Antibodies”，点击“Search”即可得到如下结果。

可以看到，页面简单的记录了抗体的几个基本信息，包括红色字体的货号和抗体公司以及抗体宿主类型（来源）、适用性、种属反应性和克隆类型（单克隆或多克隆），旁边是该抗体被引用的次数，搜索结果是根据抗体被引用次数从高到低来排列的，被引用次数越多，表明该抗体相对较为可靠。

除此之外，左边一栏可以根据抗体的一二抗类型、克隆类型、是否重组、宿主类型、验证方法、实验应用类型、种属反应性、供应商等多个目录分类查找，以便快速找到适合自己实验的抗体。

我们随机点击一个抗体进去后会有更详细的信息补充，以及最重要的是里面有应用该抗体的文献列表，可以说可靠性非常之高。
