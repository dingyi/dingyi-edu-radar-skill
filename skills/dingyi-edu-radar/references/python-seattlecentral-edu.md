# 学校介绍

西雅图中央学院是华盛顿州西雅图市的一所公立大学。它是组成西雅图大学区的三所大学之一。该学院拥有大量的国际学生，由国际教育计划部以及通过基础和过渡研究部修读ESL课程的许多移民和难民学生提供服务。西雅图中央学院还包括木材建设中心和西雅图海事学院，它们分别位于不同的校园中，以容纳所需的非常具体的工具和工作区。西雅图中心的起源可以追溯到1902年，百老汇高中成立。它一直是一所传统的高中，直到第二次世界大战结束为止，后来为了想完成高中的退伍军人的利益而改建为职业和成人教育机构。结果，在1946年，它的所有高中生全部转移到了林肯高中，而爱迪生技术学校（已经与百老汇高中共享了一个校园）被扩建为整个设施。爱迪生开始提供大学课程，当它被重组为西雅图社区学院在1966年9月北西雅图社区学院和南西雅图社区学院敞开大门在1970年，随后西雅图社区学院更名为西雅图中央社区学院。

官方网址：https://seattlecentral.edu 邮箱格式（如果申请到的话）邮箱后缀：xxxx@SeattleColleges.edu

# 密码登陆

提交之后需要等3-4天时间。你会收到一封邮件，输入临时密码，修改邮箱，记得全程开代理。

**如何获得MySeattleColleges登录名？**

在tools.seattlecolleges.edu上选择“重置”选项卡 。输入SID和PIN以及所需的密码。系统将返回您的MySeattleColleges登录名@seattlecolleges.edu。记住该名称和您输入的密码。要访问O365服务（包括电子邮件），请在Office365.seattlecolleges.edu上使用MySeattleColleges登录名和密码。

**我的电子邮件地址是什么？**

对于新生，您的Google电子邮件地址是MySeattleColleges登录名firstname.lastname @ seattlecentral.edu。初始密码是您的完整SID，您将在首次登录时进行更改。对于回国学生，您的Google邮件地址保持不变。

**如果忘记密码怎么办？**
Google应用程序现在支持通过短信或备用电子邮件地址重置密码。
