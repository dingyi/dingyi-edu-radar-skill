# 前言介绍

UNiDAYS
**是一家英国的折扣网站**
，是一套适用于全球教育的折扣商店认证体系。成立于2011年，可供全世界的学生免费使用，为学生提供快速，免费的独家优惠。 只要美国在读高等教育(高中、大学)的学生都可以通过在UNiDAYS 注册并验证，来获得产品和服务的折扣优惠。 总的来说，以前购买教育优惠是随机抽查资格，而现在100%使用上了UNiDAYS 进行身份验证，让学生用户的权益得到了保障。

UNiDAYS专门为大学生提供专属的教育折扣。不过相比之下，UNiDAYS的折扣范围更广一些，涵盖为来自阿迪达斯，三星，H＆M等150多个品牌的学生提供独家优惠。Fashion、Beauty、Lifestyle、FoodHealth&Fitness和Technology几大板块，每个板块中都有包括实体店、网店以及购物网站多种渠道的折扣信息，算得上student leap card的豪华升级版学生折扣卡了。
