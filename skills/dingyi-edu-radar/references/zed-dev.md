# 常见问题

**哪些人有资格参加 Zed 学生计划？**

目前就读于全球认可大学的学生。申请时需要验证您的入学资格。

**如果我一年内毕业呢？**

只要您在申请之日仍是该校学生，您的学生计划将保持有效一年。

**学生计划包含哪些内容？**

Zed Pro包含所有专业版功能：无限次编辑预测，以及每月10美元的AI代币额度。该套餐有效期为一年。

**如何验证我的学生身份？**

使用注册时间超过 30 天的 GitHub 帐户登录，然后分享您有效的当前大学电子邮件地址。

**我为什么需要GitHub账号？**

所有 Zed Pro 套餐（包括学生套餐）均需使用 GitHub 登录。这有助于我们验证您的身份并防止滥用。

**验证需要多长时间？**

申请通常会立即获得批准。如果您的申请需要人工审核，请发送电子邮件至education@zed.dev，我们会在 72 小时内回复您。

**一年后会发生什么？**

12 个月后，您的学生套餐将自动降级为免费套餐。您可以随时升级到 Zed Pro。

**我的申请出了问题，我应该联系谁？**

请将您的问题发送至education@zed.dev。我们会在 72 小时内回复。
