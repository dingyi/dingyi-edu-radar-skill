# 前言介绍

TryHackMe是一个在线的网络安全学习平台，旨在帮助用户从零起步学习和提升网络安全技能。它提供了一系列的虚拟实验室和挑战，供用户进行实践和学习。通过TryHackMe，用户可以学习到各种网络安全相关的知识和技术，如渗透测试、漏洞利用、密码破解、网络分析等。

**核心特点:**
游戏化学习：通过“房间（Rooms）”形式分步学习，每个房间聚焦一个主题（如基础工具使用、漏洞利用）。 实战靶场：提供在线的渗透测试环境（如 Kali Linux 虚拟机集成），支持实时攻击演练。 结构化路径：预设了多个学习路径，涵盖 Web 安全、网络攻防、逆向工程等。 社区与协作：支持多人对抗，内置讨论区与积分排行榜。

**适应人群：**
从零开始的网络安全爱好者：无需前置知识，提供了与基础渗透测试工具（如 Nmap、Burp Suite）相关的入门教程。 网络安全证书备考者：可针对性训练 OSCP 等认证考点。 网络安全教育者：支持创建学习班级，可跟踪学生课程学习进度。

需要准备一个美国edu邮箱，本次教程演示以第三方edu邮箱平台的高端美国edu教育邮箱【镇店之宝4号】 为测试邮箱演示，我们打开https://tryhackme.com/ 点击Join for FREE加入，弹出以下注册提示框：，我们直接用edu邮箱注册。
