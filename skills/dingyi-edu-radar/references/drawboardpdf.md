# 常见问题

**我可以在订阅之前试用 Pro 工具吗？**
你一定可以！转到我们的任何应用程序并选择 Pro 计划以免费试用 7 天。在 Windows 和 iOS 上，您需要输入信用卡信息，但在试用结束之前不会被扣费。

**我可以跨设备使用 Pro 工具吗？**
是的。如果您订阅 Drawboard PDF Pro 计划并创建云帐户，则无论您在何处登录同一云帐户，您的订阅都将可用。

**我是否必须创建云帐户才能使用 Pro 高级工具集？**
对于除 Store+Share 之外的所有工具，您无需云帐户即可在 Windows 和 iOS 应用程序中订阅。对于网络应用程序，您需要创建一个云帐户才能订阅。
