# 前言介绍

在当今资讯爆炸的时代，资讯的快速传播，为我们提供了庞大的学习红利，似乎想查什么资讯都唾手可得，但是，如何快速有效从海量资讯中提取有价值的内容，却成为了一大挑战。本文想分享「Consensus」这款应用，利用AI技术帮助我们在茫茫资讯海中，迅速找到关键的资料。

在不同的工作场合中，可能需要经常查询各式各样的问题，网路普遍推荐当前热门的ChatGPT，这款广受欢迎的AI聊天机器人确实很好用，不过，我个人认为，面对于某些专业领域或学术方面的问题，ChatGPT比较难以给出适切的回覆，特别是在「精确附上资料来源」这方面，即使运用ChatGPT Plus的相关插件，还是不太适用。

由于这个困扰，我开始寻找一款science-based的AI工具，我希望这款工具不仅能准确回答专业问题，更重要的是能提供引用资料来源，在一番搜寻之后，其中一款符合需求的工具就是Consensus。

Consensus这款软体奠基于庞大的学术文献资料库，它运用AI技术，分析使用者提出的问题，自动搜寻文献，并帮助我们进行摘要。

虽然Consensus与「学术」两个字挂勾时，可能让人有种距离感，好像科学家才会用到这个工具，但其实我们生活中遇到的许多问题，如果想获得比较有科学根据、或是跟Google搜寻取向不同的答案，都能尝试运用Consensus。
