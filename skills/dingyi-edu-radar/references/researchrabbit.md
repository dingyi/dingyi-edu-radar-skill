# 基本使用

我们注册成功登录后，页面空空如也，我们需要点击左侧「New Collection」，创建一个新的组，然后即可添加文献。
**点击「Add Papers」即可添加文献，可以通过「Title」、「DOI」等内容精确检索文献。**

检索时可以根据生命科学领域或全领域这两个选项进行检索，然后添加我们想要的文献。

添加好文献后在文献列表会出现该文献，点击后会在右侧弹出题目、摘要等信息，
**如果可以获取 PDF 在这里也会直接提供 PDF 的原文下载链接，点击文章题目可以直达 PubMed 的文章页。**

当我们输入一篇文献之后，可以看到文章的标题、作者、摘要，在右上角还可以看到是否有PDF能够直接下载等信息，更为重要的是，在对话框下部我们可以直接看到这篇文章中的参考文献和被引用情况，尤其是被引用情况真的是太方便了。

一个人走得快，一群人走得远。如果您有好的文章和研究思路需要和团队一起分享，那么ResearchRabbit也能让您的工作快速分享给对方。只需在对应文献集（My Collections）下面添加课题组其他成员的Email就可以了，同时可以选择对方是否有权限修改文献集（My Collections），增加或删除文献。
