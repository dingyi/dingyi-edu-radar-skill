# 注意事项和问题

1、关于教育邮箱的问题，目前网络上很多个人利用edu域名搭建域名邮箱的方式，伪装学生身份申请jetbrains产品，jetbrains目前定期会对提供的教育邮箱地址的二级域名进行真伪性核查，像网络上流行的波兰域名edu.pl和塞尔维亚国家域名edu.rs等，都是已经在黑名单系列中的。

2、jetbrains对每个教育机构都是有许可证数量限制，达到阀值就无法正常申请了。

3、授权的稳定性方面我这边网友亲测，大概半年后授权会失效，所以说网上公开免费申请的美国邮箱并不稳。

# 其它问题

因为有的买家之前是用破解的那种，破解都会修改本地文件和软件配置，那么用授权账户就会提示无法登录。

**解决办法：**

Mac上如何彻底删除Jetbrains系列的IDE

除了删除.app文件以外，还需要用命令行删除如下目录下的App名字目录即可，如Goland 2018.2版本，会存在个Goland2018.2的目录

~/Library/Preferences/

~/Library/Caches/

~/Library/Application Support/

~/Library/Logs/

把以上四个路径下的App目录都删除后。如果不会用命令行，也可以打开Finder，利用Command + Shift + G 快捷键跳转到某个目录，以此删除即可。

同理，如果想要初始化配置啥的，也可以删除对应的目录，重启即可。
