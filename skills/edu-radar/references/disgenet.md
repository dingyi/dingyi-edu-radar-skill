# 前言介绍

随着基因组学和精准医学的飞速发展，基因与
疾病
之间的关联研究已成为现代生物医学领域的重要组成部分。通过深入理解基因突变与
疾病
之间的联系，科学家和临床医生能够更加精确地预测
疾病
风险
、优化治疗方案并开发个性化的医学干预措施。在这一背景下，基因-
疾病
关联数据库的构建和应用发挥了重要的推动作用。
[DisGeNET](https://disgenet.com/)
作为全球最大的基因-疾病关联数据库之一，为基因组学研究和临床医学提供了宝贵的资源，disgenet官方网站：https://disgenet.com

[DisGeNET](https://disgenet.com/)
（Disease Gene Network）是一个综合性的基因-疾病关联数据库，专注于收集和提供关于人类基因与疾病之间关系的数据。该数据库目前包含超过628,685个基因-疾病关联信息，涉及17,549个基因和24,166种疾病/性状（截至2019年1月）。DisGeNET不仅涵盖了孟德尔遗传病、复杂疾病、环境疾病和罕见疾病的基因关联，还包括了与疾病相关的性状信息。作为一个开放性资源，DisGeNET为研究人员和临床医生提供了丰富的基因-疾病关联数据，帮助揭示不同基因在各种疾病中的作用，为精准医学的发展提供数据支持。

# 常见问题

**注册表单不接受我的电子邮件地址？**

为了申请DISGENET 免费学术许可证，您必须使用有效的机构电子邮件地址。您可能会看到以下消息： “请输入有效的机构电子邮件地址” 不接受个人电子邮件提供商，例如 gmail、hotmail qq 163等。 如果您使用的是机构电子邮件地址但仍未被接受，请联系我们。

**为什么我无法通过免费学术许可证查看所有数据？**

您的免费学术许可证可让您访问大量 精选的 疾病基因组学数据。但是，要探索 DISGENET 的全部内容（包括来自科学文献的文本挖掘数据），您需要升级到标准或高级许可证。免费学术许可证不授予您访问完整数据库的权限。要获得对所有 DISGENET 数据（包括精选和文本挖掘数据）的完全访问权限，您必须将帐户升级为标准或高级。 比较计划并了解更多信息： https://www.disgenet.com/plans

**我可以下载 DISGENET 数据库吗？**

**对于学术用户**
学术用户无法下载完整的 DISGENET 数据库。但是，您可以通过 Web 界面或 API 以各种格式（例如 TSV 和 Excel）下载搜索结果。 获取结果取决于您的计划： 免费学术计划：仅提供对精选数据的访问。 标准和高级计划：提供访问完整数据库的权限。

**对于商业公司**
营利性组织可以通过购买标准或高级许可证来访问完整的 DISGENET 数据库。搜索结果可以多种格式下载，例如 TSV 和 Excel。 如果您是一家大型企业，希望下载 DISGENET 数据并将其集成到您的工作流程中，请联系我们以了解我们的高级许可证。

**disgenet2r 403 学术许可证禁止错误?**

如果您使用具有学术许可证的 disgenet2r 并遇到访问问题，请注意： 学术许可证仅提供对“CURATED”数据库的访问权限。 database =“ALL” 将导致 403 Forbidden 错误，因为并非所有数据源都包含在学术计划中。 要解决此问题，请在函数调用中指定“CURATED”作为数据库。

**为什么我在 DISGENET 中只能看到前 30 个搜索结果？**

出现这种情况可能有以下几个原因：

1. 您注册了 7 天高级免费试用 我们的 7 天免费试用版可让您体验 DISGENET 的高级功能，包括文本挖掘数据，但有限制。如果您有试用帐户，则只能查看前 30 个搜索结果。 免费试用旨在让您有机会体验高级计划，但直到您订阅该计划后才能完全访问所有功能。

2. 您在过去 7 天内申请了免费学术许可证 您最近申请了免费学术许可证。在审核您的申请期间（通常在 7 天内），您可以暂时使用高级免费试用版，但有限制，包括只能查看前 30 个结果。 一旦获得批准，您的帐户将自动切换到免费学术许可证，并授予对精选数据的完全访问权限。 我目前有什么计划？ 要查看当前计划和查询限制，请访问https://www.disgenet.com/Tools#console（请注意，您必须登录）
**升级以访问完整的 DISGENET 数据库**
免费学术许可证或高级免费试用版不授予您访问完整数据库的权限。 要获得所有 DISGENET 数据（包括精选和文本挖掘数据）的完全访问权限，您必须将您的帐户升级到标准版或高级版。 在此比较计划并了解更多信息。
