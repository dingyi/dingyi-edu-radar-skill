# 前言介绍

[Dify](https://dify.ai/)
是一个开源的大型语言模型（LLM）应用开发平台，结合了Backend-as-a-Service 和LLMOps 的概念，帮助开发者快速构建生产级的生成式AI 应用程式。即使是非技术人员，也可以参与AI 应用程式的定义和资料操作；Dify 内建了构建LLM 应用所需的关键技术堆叠，包括支援数百种模型、直观的Prompt 编排界面、高品质的RAG 引擎以及灵活的Agent 架构，同时提供一系列易于使用的介面和API，从而为开发者节省了大量重复造轮子的时间，让他们专注于创新和业务需求。Dify官网是：https://dify.ai/

**消息积分如何运作？**
使用 GPT-3.5-turbo（默认）的 AI 响应一次需要 1 个消息积分。使用 GPT-4 的 AI 响应一次需要 20 个消息积分*。您可以在聊天机器人设置中更改聊天机器人使用的模型（目前代理助手不支持此功能）。当计划中的消息积分用完后，您可以升级版本或使用您自己的 LLM API 密钥。

*
**注意**
：由于模型成本原因，消息积分不支持 gpt-o1-preview 和 gpt-o1-mini 模型。请使用您自己的 API 密钥访问它们。
