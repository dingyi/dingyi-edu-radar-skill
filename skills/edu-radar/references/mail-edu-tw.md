# 前言介绍

台湾省教育部校园云端邮件，是由中国台湾省教育部建置，台湾全省教师、职员、学生都符合资格，透过各县市单一签入机制认证后，可以自己申请喜欢的帐号XXXXX@mail.edu.tw，无论升学或转学后还是可以使用。官方网站mail.edu.tw

# 服务对象

校园云端电子邮件信箱服务对象：包含全省公私立国小、国中学校教职员。未来将规划延伸推广服务包含高中职以上的所有教育单位。（区别于高中学生本身的学校电子邮箱）

点击申请教育云端账号，跳转到申请类型页面如下图所示：在这里为了方便给大家演示，本站选择的是学生。

选择学生后，我们来到了选择县市/行政区/学校步骤：填写我们台湾高中的学校县市行政区，然后就会出来学校选择，我选的是淡江高中。

# 个人验证

我们来到了个人资料验证页面，输入内容要按照台湾省的文字格式和纪年法格式，姓名繁体，生日要按照西元年格式去填写。

# 账号设置

等到我们的个人验证通过后，就来到了教育账号设置页面，就是可以设置一个自定义用户名。

帐号长度须大于6个字元，第1个字元为英文字，第二个之后的字元可为”英文/数字/底线/减号/点” (但不能有特殊符号，如!@#$%^&*()等)
