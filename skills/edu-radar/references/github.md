# 必备条件

需要一个教育邮箱，github对每个高校的申请数量都有很严格的限制，由于这几年国外教育邮箱的滥用，大多数高校都已经无法正常秒申请github

# 注册github账号

登陆github官网地址：https://github.com/

1、点击右上角“Sign Up”

在“Username” 创建登陆用户名，在这里我便于方便和记忆，直接输入的是教育邮箱登陆名。

在“Email address”输入你的教育邮箱。在“Password”至少使用一个小写字母，一个数字和七个字符。

点击创建。

2、选择一个计划

选择免费的无限公共仓库并Continue。

3、选择你的经验 直接跳过即可

打开教育邮箱收件箱，会收到github的确认邮件，点击确认即可，此步骤验证为验证github账户，非学生资格。

# 验证学生身份

打开github学生包验证网址：https://education.github.com/

点击：“Get the pack”

选择YES，I’m a student

GitHub Student Developer Pack只对13岁以上的学生开放。在您收到录取通知书之前，我们需要确认您是一名学生教师、研究人员、教职员工和其他教育用户可以免费获得GitHub的折扣访问，但不符合该包的条件。如果你不是学生，你仍然可以申请定期的GitHub教育折扣。

“Verify academic status” 验证你的学术邮箱 “School name” 输入你的大学名字

“Graduation year” 输入你的毕业年限，（不要乱填，填相差四年就够了）

“How do you plan to use Github” 随便复制点英文内容即可。

# 申请提交

提交完信息后，五天之内会收到回复邮件，但是一般2分钟之内就可以收到，如果学校没有被滥用屏蔽的话。

# 申请成功

我们已经为您升级到一个计划，提供无限制的免费私有存储库，该计划将在未来两年内免费。之后，你会收到一封电子邮件，说你的优惠券到期了。如果你仍然有学术地位，你可以重新申请另一张优惠券。我们没有任何协作限制，所以您可能遇到的任何团体项目都可以通过您的帐户托管。
