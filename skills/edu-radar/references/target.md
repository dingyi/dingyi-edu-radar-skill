# 常见问题

**我如何验证我的学生身份？**

Target 应用程序– 转到“我的目标”选项卡，点击右上角的“设置”（轮子图标），然后选择“您的折扣”部分下的学生验证。在这里您可以输入您的信息并提交验证表。

Target.com – 登录您的帐户，从“Hi”问候语下的下拉菜单中 选择“帐户” 。在您的 Target 帐户中选择“设置”，然后在“折扣”部分中选择“验证状态”，然后在“学生”部分中选择“验证状态”。在这里您可以输入您的信息并提交验证表。

**谁有资格享受 Target 的学生折扣？**

Target Circle™ 会员是目前就读于授予学位的高等教育学院或大学的大学生。

**为什么我没有立即得到验证？**

您的验证可能由于常见问题（例如使用昵称或输入信息时出现拼写错误）而失败。请检查您的所有信息，确保输入正确，并与您的官方文件上显示的一致。 常见问题可能包括： 文件上显示的姓名不同。例如，官方文件上列出的名字为“Jamie”，但验证表上提交的名字为“Jay”。 常见拼写错误（例如输入错误的出生年份）

**我为什么还没有收到折扣？**

当有实时活动进行时，您的 Target Circle 奖金将出现在您的帐户中。成功验证学生身份后， 请等待最多 1 小时。 如果您的奖金仍然没有出现，请检查您的帐户以确保您的验证成功。 如果您在通过 SheerID 验证时遇到问题，请访问他们的帮助中心。 如果您已经验证并加入 Target Circle 但仍找不到您的奖金，请联系 Target 的帮助中心。
