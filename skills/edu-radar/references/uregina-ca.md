# 学校介绍

里贾纳大学位于萨斯喀彻温省省府里贾纳市。大学的前身可以追溯到1911年成立的里贾纳学院，这是一所由加拿大卫理公会建立的私立社区中学，最初只招收了27名学生。在其发展过程中，它曾一度成为萨斯喀彻温大学的第二个校区，后来经过不断的发展与合并，直到1974年里贾纳学院才成为一所可以独立授予学位的学院。目前里贾纳大学拥有学士、硕士和博士学位授予权。

经过三十多年的发展，大学树立了自己良好的形象。现在的里贾纳大学已成为一所拥有9个学院、25个学术科系、13个研究中心的综合性大学。

正是由于学校位于萨斯喀彻温省的首府，因此该校课程设置均为适应萨省的需求而定，在人文、记者专业等方面尤为突出。近几年随着政府和工业的发展，理学和工程专业也得到了长足的进步。在公共服务专业和石油工程专业吸引了越来越多的学生和国际的关注。从 1992 年起，学校从事研究方面的资金从340万增加到1,200万。学校同时也是国家几个从事智能机器人技术和智能系统检测的网络中心之一。学校也与其它几个国家级的研究中心合作。

# 创建别名

**通过创建辅助别名地址来自定义uregina.ca电子邮件帐户：**

- 发送到别名地址的电子邮件将传递到您通常的uregina.ca电子邮件帐户。
- 如果您的电子邮件转发到另一个地址，别名也将自动转发到该地址。
- 别名 只能包含字母和数字，不能包含空格，句点或其他字符。
- 别名 不能匹配任何现有的用户名。
- 员工帐户为
[first.last@uregina.ca](mailto:first.last@uregina.ca,)
，只有该帐户才能具有匹配的
[firstlast@uregina.ca](mailto:firstlast@uregina.ca)
别名。
- 每个帐户仅允许使用一个额外的用户别名。
- 包含冒犯性语言，旨在冒充或欺骗他人或以其他方式被视为不当行为的别名将被删除，并可考虑采取进一步的纪律处分。
- 别名必须遵守
[尊重大学政策](https://www.uregina.ca/policy/browse-policy/policy-GOV-100-015.html)
。

[在此处登录以创建电子邮件别名](https://accountapps.uregina.ca/cgi-bin/alias/addalias.pl)
