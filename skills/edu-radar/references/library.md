# 前言介绍

学术数据库资源本身是互联网时代英美学术图书馆分享知识的努力，互联网真正的分享是没有学历门槛的，但目前这种分享仍主要局限于大学校园内，当网络百科的威力超过任何百科全书时，所谓“非营利资源库”依然以费用的高门槛拒绝学校机构外的个人用户查询论文，虽然哈佛、耶鲁、麻省理工的网络公开课让全世界的网名在自己的手机和电脑中有机会倾听大师的精彩讲课，但保守的“分享但以大学物理围墙为界”原则，的确是在阻止互联网上真正的知识分享。互联网使得大学的“物理围墙”真正被打破。
我们在家也能足不出户的享受到大洋彼岸美国高等学府图书馆的电子教育资源。

有很多用户对国外数据库资源不太了解，认为edu邮箱没有什么用处，喜欢用电商短期的虚假edu邮箱，结果到手也用不了，但对很多学术大佬和下载文献的群体很重要，无论是某多某鱼某宝，都没有这类产品，因为商家可能自己都不知道有图书馆这类功能，所以市场上这类东西几乎没有。

今天我们就以
**享誉【镇店之宝3号】高端美国edu教育邮箱A型**
的数据库分类进行展示，截止20240620以下数据库状态均为该学校目前拥有数据库数量，如遇到变化请以登录校方图书馆刷新实际状态为准。按字母排列共计
**232种著名海外图书馆。**

以下图是以翻译后的显示：我们看到文献数据库展示是以数据库名称首字母的24个英文字母排列，我们点击显示所有描述，也会有数据库的重点介绍。

为了方便师生分类使用，也按照数据库类型进行分类，我们看到涵盖了各行各业：艺术、商业计算机和经济、国家和文化、数据库集合、教育和职业、电子书和杂志，百科全书和词典、政府、健康与护理、历史和法律资源、文学和文学解读、多语言资源、多学科、新闻、心理学、哲学、宗教和行为科学、统计数据等。

# 人性化功能

有很多用户对国外数据库使用比较陌生，学校官方也提供了图书馆使用教程，在图书馆首页都可以很醒目的看到，而且还有，全天候为学生提供免费的在线技术帮助。要获取此帮助，请使用以下链接。这包括 Blackboard 帮助、密码和登录帮助、远程访问等。 为学生提供的虚拟桌面基础设施 (VDI) 服务现已推出。VDI 允许学生从任何可以访问互联网的地方登录 Windows 10 虚拟桌面。使用 VDI 就像登录校园计算机一样，但可以让学生远程访问校园计算机上常见的许多应用程序。

# 演示体验

以下学术视频、学术文献下载、电子书、图书馆借书等学校的所有功能服务，均需要在edu邮箱登录状态下进行，因为无论是学校自己付费购买了这些资源，所以每一步都需要用edu邮箱鉴权后，比如打开数据库第一步，数据库先要看身份的资格是否符合，符合后才跳转到数据库使用页面，才可以访问其他图书馆资源。
