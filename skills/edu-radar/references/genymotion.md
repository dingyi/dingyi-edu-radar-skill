# 前言介绍

GenyMotion 是一款比官方 Android 模拟器还快的模拟器，Genymotion 提供云端和PC、Mac上的Android模拟器服务。主要产品包括Genymotion SaaS（云端Android服务）、Genymotion Device Image（云服务和本地Android镜像）和Genymotion Desktop（适用于Windows、MacOS和Linux的Android模拟器）。服务对象涵盖企业、个人、学校等，支持移动测试开发和云端Android应用。Genymotion 提供了多个版本和组件，主要分为 ​
**​Genymotion Desktop​**
​（本地版）、​
**​Genymotion SaaS​**
​（云端版）和 ​
**​Genymotion Device Images​**
​（设备镜像）。它们的核心区别在于 ​
**​运行方式、使用场景和功能定位​**
​。

# 版本比较

官方价格介绍：https://www.genymotion.com/pricing/#edu-order

**Genymotion Desktop​​（本地版）**

安装在个人电脑上的本地安卓模拟器，适合单机使用，依赖本地虚拟机（VirtualBox），需下载并存储在电脑上​​，适用人群​​：开发者、测试人员、普通用户（免费版有使用限制）。

**免费版本：**
个人使用，仅有少数小部件和功能可用，Android 的最新版本不可用，没有提供技术支持。

**教育版本：**
适用于学生、教师和学校。需提供有效证明。49美金/年；免费小部件，专业小部件，最新的 Android 版本，尽最大努力电子邮件支持。

**专业版本：**
面向自由职业者或个体经营者，206美金/年；免费小部件，专业小部件，最新的 Android 版本，高优先级技术援助的电子邮件支持。

**商业版本：**
适用于承包商和经销商。412美金/年；免费小部件，专业小部件，最新的 Android 版本，VIP 提供 SLA 技术援助的电子邮件支持。

**Genymotion SaaS​​（云端版）**

基于云服务的安卓虚拟设备，无需本地安装，通过浏览器访问；设备运行在Genymotion的云端服务器上；适用人群​​：企业团队、需要远程协作或跨平台测试的开发者。

**随用随付：**
$0.05/分钟 & 设备* 仅按实际使用量付费 *适用于 x86/x86_64 虚拟设备。

**企业版：**
$150/月 & 设备* 每年计费，无限使用且费用可预测，高级技术援助 *适用于 x86/x86_64 虚拟设备。

**Genymotion Device Images​**
​（设备镜像）

预配置的安卓系统镜像文件，用于快速创建虚拟设备；需配合Genymotion Desktop或SaaS使用，本质是“系统模板”；需要特定安卓版本或设备型号的开发者；提供官方优化过的镜像（如Android 13 + Google Pixel 6配置）

# 常见问题

**Genymotion Desktop 的要求是什么？**

Genymotion Desktop 在 Windows 和 Linux 上需要 x86_64 CPU。因此，它不适用于基于 ARM64 的计算机和单板，以及 Windows 和 Linux。要使用 Genymotion Desktop，必须拥有 Genymotion 帐户。您可以启动 Genymotion Desktop，并在系统提示创建帐户时单击
**“创建” ，也可以转到**
[此处](https://www-v1.genymotion.com/account/create/)
。

首次运行并下载虚拟设备映像时，必须连接互联网才能登录。但是，登录后会生成一个有效期为 15 天的令牌，因此可以在每次连接之间的 15 天内离线使用 Genymotion Desktop。但在此之后，您必须连接到互联网，以便 Genymotion 可以验证您的凭据。

支持的 Windows 版本 Genymotion Desktop 已针对以下 Windows 版本进行设计和测试： Windows 10 或 11（64 位）

不受支持的系统 – Windows 32 位 – Windows XP、Vista、7、8 和 8.1

Linux 支持的 Linux 发行版 Genymotion Desktop 已在以下发行版上进行了全面测试： Ubuntu 24.04LTS Fedora Workstation 40（64 位） Debian 12 Bookworm（64 位） 其他发行版（例如 Kali Linux 或 Arch Linux）未获得官方支持。

**如何在 Genymotion 中安装 Google Play Store（和其他 Google Apps）？**

虚拟设备上未预装 Google 应用和服务。您需要刷新 Open GAPPS 包才能获取它们。
[安装教程](https://support.genymotion.com/hc/en-us/articles/4414586104977-How-to-install-Google-Play-Store-and-other-Google-Apps-in-Genymotion)
