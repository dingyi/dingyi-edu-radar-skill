# 身份验证

**SheerID如何验证您是一名大学生**

在美国，有一个名为“国家学生信息交换所”的“非营利性组织”，它具有有关在美国大多数大学中就读的所有学生的就读信息。他们有一个称为验证服务的产品，该产品可让第三方验证这些学生的入学情况。

交换所（他们喜欢自称为）是由大学组成的。大学是其董事会成员。从本质上讲，这些大学“拥有”票据交换所。最初是为了使大学可以交换学生的成绩单而设置的。当学生从一所大学转到另一所大学时，他们必须发送成绩单。上世纪70年代，这是一个复杂的过程。每所大学都会向所有其他大学发送成绩单。伪造您的成绩单也很容易。因此，他们创建了票据交换所。大学将其注册信息发送到信息交换所。无论何时，该学院需要成绩单入学验证，只需向信息交换所询问即可，然后免费获得。事实证明，让票据交换所这样做更便宜，因为这只是一个大数据库。

**验证过程如何进行**

sheerid客户（提供折扣的品牌）确定谁有资格进行促销。然后，sheerid创建一个表格，您可以在其中仅输入确认您合格的信息。提交信息后，会将其与选定的权威数据源进行比较，而不会使任何人看到数据。我们会立即确认是否存在匹配项，然后返回“是”（表示触发奖励交付），或者返回“否”（表示您不符合条件或您需要提供更多信息）。

# 常见问题

**如果被问到，我应该上传什么样的文件？**

您需要上传由学校发布的文档，以显示您的姓名、学校名、显示当前注册的日期

请注意，我们将审查注册日期，而不是打印日期或查看日期。例如，您的成绩单可能是昨天打印的，但是它表明您最近一次上课是上学期。将来的条款也不被接受，因此录取通知书不能证明当前的入学人数。

**我刚刚上传了一个文档。我什么时候会收到要求的结果？**

文件按照收到的顺序进行检查。在接下来的48小时内，将通过电子邮件与您联系，并提供有关您的验证请求的其他详细信息。如果收件箱中没有电子邮件，请检查您的垃圾邮件/垃圾文件夹。该电子邮件将告知您是否已成功通过验证，或者是否需要其他信息来完成验证。

**如果我不能证明自己是学生会怎样？**

这是极为罕见的。学术机构提供了我们可以接受的多种形式的官方文档，但是如果您无法向我们提供任何当前就读的可接受证明，我们将无法批准您的提议。但是要重复一遍，如果您正在上学，则应该可以访问我们可以验证的文件。

如果您的学校不在列表中，那么您目前不符合此条件。为确保您没有错过列表中的学校，请输入学校的全名。
