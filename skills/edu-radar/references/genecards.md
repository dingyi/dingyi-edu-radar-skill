# 前言介绍

今天也是网友来本站寻求edu邮箱帮助，需要注册一个学术网站，本站立马精神起来，问小姐姐注册什么学术网站呀，小姐姐就发来网址一看好专业好厉害的感觉，就自己尝试注册了下，没想到就成功了，当然也不知道是什么用，但是很开心能够给大家分享教育资讯的线报。

[GeneCards](https://www.genecards.org/)
由非盈利组织构建的一个整合型的生物信息数据库，该数据库提供了人力目前已注释的、可预测的所有基因的详细信息，并自动集成了来自约150个数据源的以基因为中心的数据，包括基因组、转录组、蛋白质组、遗传学、临床和功能信息。

# 信息填写

这一步是为了校验学术邮箱的真实性，如果您提供的edu邮箱是真实的大学邮箱，GeneCards会自动帮您识别出学校全称。只要识别出来，别的就随便填了啦，当然为了真实有效，我们在“
**Department**
”部门填写“Bioengineering Institute”生物工程学院的意思，可以用别的专业反正是英文就可以了。如下图所示：

Country:国家，Organization: 学校名称，Department：院系名称，Job Title:角色，

Where did you hear about us? 你从哪里知道我们的，随便填，Phone Number:手机号，填写时候全部用英文。

我们点击Join Now 进入下一步，就来到了账户确认页面，输入edu邮箱里面的数字验证码。
