# 前言介绍

[Wmatrix](https://ucrel.lancs.ac.uk/wmatrix/)
是一个语料分析和比较的软件工具。它为自然语言处理工具提供了一个 Web 界面，例如
[USAS](https://ucrel.lancs.ac.uk/usas/)
和
[CLAWS](https://ucrel.lancs.ac.uk/claws/)
英语语料库注释工具，以及（从 Wmatrix6 开始）
[多语言语义标记器 PyMUSAS](https://pypi.org/project/pymusas/)
以及标准语料库语言方法（例如频率列表和索引）。它将关键词方法扩展到关键语法类别和关键语义领域。

Wmatrix 允许用户通过 Chrome 或 Firefox 等网络浏览器运行这些工具，因此可以在任何具有网络浏览器和网络连接的计算机（Mac、Windows、Linux）上运行。
[Wmatrix 最初由Paul Rayson](http://www.lancaster.ac.uk/staff/rayson/)
在
[REVERE 项目](http://www.comp.lancs.ac.uk/computing/research/cseg/projects/revere/)
中开发，在博士期间扩展并应用于语料库语言学，并且仍在定期更新。早期版本可通过基于终端的命令行访问 (tmatrix) 适用于 Unix，也可通过 Xwindows (Xmatrix) 适用于 Unix，但这些版本仅提供对预先用 USAS 和 CLAWS 注释的文本的检索。
